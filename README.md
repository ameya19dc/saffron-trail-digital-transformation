# saffron-trail-digital-transformation
This project is a simulated consulting engagement where I play the role of the consultant who helps the spice manufacturing and trading company, Saffron Trail Spices Pvt. Ltd., digitalize their systems of working, and claude.ai creates the virtual company environment and roleplays the employees and executives. It also plays the role of my mentor since this is a first of a kind project for me.

Context: The goal is to learn skills for doing this in a real life setting as fast as possible while working on my full-time job as a chef.
