# Simulation Time Convention

## The Convention
1 day deadline in simulation = 5 hour deadline in real time

## Why
- It almost matches a real consultants hours but is still lesser so I don't drag the project too long.

## How To Apply It
- The 5 hour time is the cap. 
- If the work finishes faster then we finish faster and are ahead on the project.
- If a task takes longer than expected then I should adjust the scope next time but if a commitment is made and I take extra time, then I will log it as extra time taken which would be similar to having spent extra time in real life to keep up with the commitment and then adjust how to make commitments in the future.

## When To Revisit
Not anymore in this project.
